# Blast7antakorr

## Setup Instructions
1. **Clone the Repository**
   ```bash
   git clone https://github.com/Just-brownie/Blast7antakorr.git
   cd Blast7antakorr
   ```

2. **Install Dependencies**
   Make sure you have Python installed. Create a virtual environment and install the required packages:
   ```bash
   python -m venv env
   source env/bin/activate  # On Windows use `env\Scripts\activate`
   pip install -r requirements.txt
   ```

3. **Configure Environment Variables**
   Ensure you have a `.env` file with the following variables set:
   - `API_KEY`: Your API access key.
   - `DATABASE_URL`: Your database connection string.

4. **Run the Application**
   Start the application:
   ```bash
   python main.py
   ```
